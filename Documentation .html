LDU Factorization Calculator

Introduction

Overview

This website allows users to input a matrix of arbitrary dimension and calculate its LDU factorization. The website will display the factorization process step by step, showing the L, D and U matrices that result from the decomposition.

Purpose

The purpose of this project is to help users understand the concept of matrix factorization, specifically LDU decomposition, by providing a clear and interactive tool for performing the factorization step by step.

Technologies Used

HTML

HTML is used to structure the webpage, creating input fields, buttons and a display area for the results.

CSS

CSS is used for styling the website, ensuring that it is visually appealing and user-friendly.

JavaScript

JavaScript handles the logical part of matrix input, factorization calculations.

Git

Git is used for version control, allowing team members to collaborate effectively and track changes throughout the development process.

LDU Factorization Explanation

What is LDU Factorization?

LDU factorization decomposes a square matrix A into three matrices:

  • L = Lower Triangular Matrix
  • D = Diagonal Matrix
  • U = Upper Triangular Matrix

    • A = L * D * U.

    Why is it Important?

    LDU factorization is widely used in numerical methods, solving systems of linear equations, and matrix operations.

    Input Method

    The user can take any numerical value as input min=2 and max=10 for matrix dimension and it will give a matrix of respective dimension by clicking Generate Matrix button. Then, click the Factorize button .
